📖 项目介绍

什么是 Cloudflare Tunnel 优化版？

本项目是对 Cloudflare 官方工具 cloudflared 的优化封装，通过跨平台一键脚本、智能连接优化、完善的配置模板，让用户以最简单的方式享受企业级内网穿透服务。

为什么选择本项目？

官方工具功能强大但使用门槛较高，需要手动下载安装、记忆命令参数。本项目提供一键安装启动脚本，自动检测系统、安装依赖、引导配置，让新手也能快速上手。

🔧 优化说明

📦 一键安装启动脚本

跨平台支持 macOS/Linux/Windows，自动检测系统架构，自动安装 cloudflared，交互式引导配置，后台运行支持。

🌐 IPv4/IPv6 智能兼容

使用 --edge-ip-version auto 参数，自动检测网络环境，选择最优连接方式，解决部分环境连接失败问题。

🔄 连接稳定性增强

连接重试 5 次、超时 30 秒、TCP 保活 30 秒、HTTP/2 支持，确保隧道稳定运行。

📊 状态检查与更新

提供状态检查脚本查看运行状态，更新检查脚本自动检测并安装新版本。

❓ 常见问题

快速隧道的域名会变吗？ ▼

是的，Quick Tunnel 生成的域名是临时的。如需固定域名，请使用命名隧道或自定义域名模式。

免费版本有什么限制？ ▼

✓ 无限隧道数量 ✓ 无限流量 ✓ 支持自定义域名 ✓ DDoS 防护。仅 Zero Trust 用户数限制 50 人。

如何绑定自定义域名？ ▼

1. 将域名添加到 Cloudflare 并修改 NS 服务器
2. 运行 cloudflared tunnel login 登录
3. 运行 cloudflared tunnel create my-tunnel 创建隧道
4. 配置 config.yml 并运行

支持哪些协议？ ▼

支持 HTTP/HTTPS、SSH、RDP、TCP/UDP 等。HTTP 服务可直接使用，SSH/RDP 需配置 Cloudflare Access。
